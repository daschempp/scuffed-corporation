01 — what we do

We built it, and we use it

Everything here is self-hosted: it runs on hardware we own, stores data in a database we control, and never reports back to a growth team that doesn’t exist. The scope is narrow on purpose — one thing, done all the way, beats ten things done up to the launch.

There’s no roadmap written for investors and no feature waiting on a funding round. Everything ships to production standards, because the first and most demanding user is the person who wrote it — and he’s right here.

Scuffed OS

A self-hosted personal operating system: one Claude-powered assistant with read-and-write tools over your entire life.

Tasks with reminders, recurrence, and file attachments. Calendar. Habits. Nutrition with USDA food-database lookup. A persistent second-brain memory that actually remembers — across conversations, weeks, and topics. Talk to it by text or voice; reminders fire as native notifications, not as guilt in an inbox.
