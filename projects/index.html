01 — flagshipstatus: in active development

Scuffed OS in active development

A self-hosted personal operating system: one Claude-powered assistant with read-and-write tools over every domain of a life.

Most assistant products can tell you things. Scuffed OS can do things. The assistant holds real read-and-write tools over tasks — with reminders, recurrence, and file attachments — plus calendar, habits, and nutrition with USDA food-database lookup, all backed by a persistent second-brain memory that carries context across conversations, weeks, and topics. Conversations stream and persist. You can talk to it by voice. Reminders fire as native notifications instead of dying quietly in a tab.
